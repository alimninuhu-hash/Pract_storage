Основные возможности системы

- Управление товарами:
1) Добавление новых позиций
2) Редактирование существующих записей
3) Удаление товаров
4) Поиск по названию
5) Фильтрация по категориям
6) Функционал сортировки:
7) По названию (↑/↓)
8) По количеству (↑/↓)
9) По цене (↑/↓)
10) По дате поставки (↑/↓)
11) По категории (↑/↓)
12) Автоматические расчеты:
13) Подсчет количества товаров
14) Общее число единиц на складе
15) Расчет общей стоимости склада

Используемые технологии

- Программирование:

1. C#

2. .NET

3. WPF

- Интерфейс:

1. XAML

- База данных:

1. SQL Server

Структура интерфейса

- Главное окно:

1. Таблица товаров

2. Панель управления

3. Статистика склада

4. Инструменты поиска и фильтрации

- Окно управления:

1. Добавление/редактирование товаров

2. Валидация данных

3. Сохранение в базу

Структура базы данных

Таблица товаров:

    SELECT TOP (1000) 
    [Id],   
    [Name],
    [Category],  
    [Quantity], 
    [Price], 
    [TotalPrice],
    [Supplier],
    [DeliveryDate]
    FROM [Tovary].[dbo].[Product]

Таблица пользователей:

    SELECT TOP (1000) 
    [Id],
    [Login],
    [Password]
FROM [Tovary].[dbo].[Users]

Система защиты данных

Реализована проверка:

- Пустых полей

- Отрицательных значений

- Формата цены

- Формата количества

- Корректности даты

Настройка подключения к БД

Необходимо изменить строку подключения в файлах:

- MainWindow.xaml.cs

- AddEditWindow.xaml.cs

Пример строки подключения:

       string connectionString = @"Server=YOUR_SERVER;
       Database=Tovary;
       Trusted_Connection=True;";
