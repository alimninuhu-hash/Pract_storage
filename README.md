# Pract_storage
практическая работа по МДК 02,02(Нинуху А. Колмыченко Д. Шапошникова Д.)
Возможности: Добавление товаров; Редактирование товаров; Удаление товаров; Поиск по названию; Фильтрация по категориям;
Сортировка: по названию; по количееству; по цене; по дате поставки; по категории
Автоматический подсчет: количества товаров; общего числа единиц; стоимости склада; Валидация пользовательского ввода; Современный интерфейс
Используемые технологии
C#
WPF
.NET
SQL Server
XAML
Интерфейс: Главное окно; таблица товаров; панель управления;статистика склада; поиск и фильтрация; Окно добавления/редактирования; ввод информации о товаре; проверка корректности данных; сохранение в базу данных
Структура базы данных
SELECT TOP (1000) [Id]
      ,[Name]
      ,[Category]
      ,[Quantity]
      ,[Price]
      ,[TotalPrice]
      ,[Supplier]
      ,[DeliveryDate]
  FROM [Tovary].[dbo].[Product]

SELECT TOP (1000) [Id]
      ,[Login]
      ,[Password]
  FROM [Tovary].[dbo].[Users]

Поддерживается сортировка:
Название ↑ ↓
Количество ↑ ↓
Цена ↑ ↓
Дата ↑ ↓
Категория ↑ ↓
